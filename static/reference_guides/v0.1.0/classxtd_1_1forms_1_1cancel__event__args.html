#include <cancel_event_args.h>

Definition

Provides data for a cancelable event.

Namespace
xtd::forms
Library
xtd.forms
Remarks
A cancelable event is raised by a component when it is about to perform an action that can be canceled, such as the closing event of a form.

Inherits xtd::event_args.

Inherited by xtd::forms::do_work_event_args, and xtd::forms::form_closing_event_args.
