xtd::native Namespace Reference

Definition

The xtd::native namespace contains internal native API definitions to access underlying operating system components used by xtd.core library.

The xtd::forms::native namespace contains internal native API definitions to access underlying toolkits used by xtd.forms library.

Warning
Internal use only
