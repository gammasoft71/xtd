xtd_collection_indexer.cpp

Shows how to use indexing operators with xtd::bpos (bpos, bpos + 1, bpos + 2), xtd::epos (epos, epos - 1, epos - 2, ...) and bitwise operators (~1_z, ~2_z, ...) with xtd collections like xtd::collections::generic::list.
