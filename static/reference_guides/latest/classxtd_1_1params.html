Definition

template<typename type_t>
class xtd::params< type_t >

An object of type xtd::params <type_t> is a lightweight proxy object that provides access to an array of objects of type const type_t (that may be allocated in read-only memory).

Namespace
xtd
Library
xtd.core
Remarks
This class cannot be inherited.
A xtd::params object is automatically constructed when:
  • A brace-enclosed params is used to initialize an object, where the corresponding constructor accepts an xtd::params or std::initializer_list parameter,
  • A brace-enclosed params is used as the right operand of assignment or as a function call argument, and the corresponding assignment operator/function accepts an xtd::params or std::initializer_list parameter.
  • A brace-enclosed params is bound to auto, including in a ranged for loop.
xtd::params may be implemented as a pair of pointers or pointer and length. Copying a xtd::params does not copy the backing array of the corresponding params.
xtd::params is exactly The same as std::initializer_list object.
