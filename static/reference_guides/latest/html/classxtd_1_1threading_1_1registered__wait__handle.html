bool xtd::threading::registered_wait_handle::unregister ( )

Cancels a registered wait operation issued by the xtd::threading::thread_pool::register_wait_for_single_object method.

Returns
bool true if the function succeeds; otherwise, false
Remarks
If wait_object is specified, it is signaled only if the xtd::threading::registered_wait_handle is successfully unregistered. If a callback method is in progress when Unregister executes, wait_object is not signaled until the callback method completes. In particular, if a callback method executes Unregister, waitObject is not signaled until that callback method completes.
