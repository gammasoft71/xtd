xtd::drawing::brush Class Reference
libraries » xtd.drawing

#include <brush.h>

Definition

Defines objects used to fill the interiors of graphical shapes such as rectangles, ellipses, pies, polygons, and paths.

Namespace
xtd::drawing
Library
xtd.drawing
Remarks
This is an abstract base class and cannot be instantiated. To create a brush object, use classes derived from xtd::drawing::brush, such as xtd::drawing::solid_brush, xtd::drawing::texture_brush, and xtd::drawing::drawing2d::linear_gradient_brush.

Inherits xtd::object.

Inherited by xtd::drawing::drawing2d::hatch_brush, xtd::drawing::drawing2d::linear_gradient_brush, xtd::drawing::solid_brush, and xtd::drawing::texture_brush.
