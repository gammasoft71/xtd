#include <paint_event_args.h>

Definition

Provides data for the paint event.

Namespace
xtd::forms
Library
xtd.forms
Remarks
The paint event occurs when a control is redrawn. A paint_event_args specifies the graphics to use to paint the control and the clip_rectangle in which to paint.
