xtd::drawing::colors Class Referencefinal
libraries » xtd.drawing | drawing

#include <colors.h>

Definition

colors for all the standard colors. This class cannot be inherited.

Namespace
xtd::drawing
Library
xtd.drawing
Remarks
The color objects returned by this class are immutable, meaning their properties cannot be changed.
Examples
colored_tab_pages.cpp.

Inherits xtd::static_object.
