The following table lists the encodings supported by xtd. It lists each encoding's code page number and the values of the encoding's xtd::text::encoding_info::name and xtd::text::encoding_info::display_name properties. A check mark in the xtd framework support indicates that the code page is natively supported, regardless of the underlying platform. For xtd framework, the availability of other encodings listed in the table depends on the operating system.
