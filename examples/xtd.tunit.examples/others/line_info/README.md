# line_info

Shows how to use [xtd::diagnostics::stack_frame](https://gammasoft71.github.io/xtd/reference_guides/latest/classxtd_1_1diagnostics_1_1stack__frame.html) class and [csf_](https://gammasoft71.github.io/xtd/reference_guides/latest/group__keywords.html#gaf8b977f43d229f177b7b399c3b3a60a6) keyword.

## Sources

[src/line_info.cpp](src/line_info.cpp)

[CMakeLists.txt](CMakeLists.txt)

## Build and run

Open "Command Prompt" or "Terminal". Navigate to the folder that contains the project and type the following:

```cmake
xtdc run
```
