* [src/string_valid_does_not_end_with.cpp](src/string_valid_does_not_end_with.cpp)

## Build and run

Open "Command Prompt" or "Terminal". Navigate to the folder that does_not_end_with the project and type the following:

```cmake
xtdc run
```
