# Text examples

[This folder](.) contains strings examples used by [Reference Guide](https://gammasoft71.github.io/xtd/reference_guides/latest/) and more.

* [string_builder_append](string_builder_append/README.md) shows how to use [xtd::text::basic_string_builder::append](https://gammasoft71.github.io/xtd/reference_guides/latest/classxtd_1_1text_1_1basic__string__builder_a01a81c0e4a6288fe54b854c98ad485b8.html#a01a81c0e4a6288fe54b854c98ad485b8) method.

## Build and run any project

Open "Command Prompt" or "Terminal". Navigate to the folder that contains the project and type the following:

```shell
xtdc run -t any_project_name
```
