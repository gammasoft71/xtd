# generic_params

Shows how to use [xtd::collections::generic::params](https://gammasoft71.github.io/xtd/reference_guides/latest/classxtd_1_1collections_1_1generic_1_1params.html) class.

## Sources

[src/generic_params.cpp](src/generic_params.cpp)

[CMakeLists.txt](CMakeLists.txt)

## Build and run

Open "Command Prompt" or "Terminal". Navigate to the folder that contains the project and type the following:

```cmake
xtdc run
```
