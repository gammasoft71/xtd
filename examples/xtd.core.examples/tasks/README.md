# Tasks examples

[This folder](.) contains format examples used by [Reference Guide](https://codedocs.xyz/gammasoft71/xtd/) and more.

* [tasks](tasks/README.md) shows how to use [xtd::threadin::tasks:task](https://gammasoft71.github.io/xtd/reference_guides/latest/classxtd_1_1threading_1_1tasks_1_1task_3_01void_01_4.html) class.
* [task](task/README.md) shows how to use [xtd::threadin::tasks:task](https://gammasoft71.github.io/xtd/reference_guides/latest/classxtd_1_1threading_1_1tasks_1_1task_3_01void_01_4.html) class.
* [task_delay](task_delay/README.md) shows how to use [xtd::threadin::tasks:task::delay](https://gammasoft71.github.io/xtd/reference_guides/latest/classxtd_1_1threading_1_1tasks_1_1task_3_01void_01_4.html) method.

## Build and run any project

Open "Command Prompt" or "Terminal". Navigate to the folder that contains the project and type the following:

```shell
xtdc run -t any_project_name
```
