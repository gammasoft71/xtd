# linq_from4

Shows how to use [xtd::linq::from](https://gammasoft71.github.io/xtd/reference_guides/latest/group__linq.html#ga4f77c7ad6f2b2bb5c560d46f822b85df) method.

## Sources

* [src/linq_from4.cpp](src/linq_from4.cpp)
* [CMakeLists.txt](CMakeLists.txt)

## Build and run

Open "Command Prompt" or "Terminal". Navigate to the folder that contains the project and type the following:

```cmake
xtdc run
```
