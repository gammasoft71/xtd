Shows how to use [xtd::linq::enumerable::all](https://gammasoft71.github.io/xtd/reference_guides/latest/classxtd_1_1linq_1_1enumerable.html#a9fcb75c4192ead30fe619401d8455c80) method.

```cmake
xtdc run
```
